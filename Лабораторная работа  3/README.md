# Лабораторная работа 3 - Автозагрузчик классов

## Описание

Реализация автозагрузчика классов с использованием PSR-4 стандарта и создание веб-интерфейса для визуализации данных.

## Требования

- PHP >= 7.4
- Composer

## Установка

1. Установите зависимости через Composer:
```bash
composer install
```

## Запуск

1. Запустите встроенный PHP сервер:
```bash
start-server.bat
```

Или вручную:
```bash
php -S localhost:8000
```

2. Откройте браузер и перейдите по адресу: http://localhost:8000

## Структура проекта

```
Лабораторная_работа_3/
├── src/
│   └── Models/
│       ├── User.php       # Класс пользователя
│       ├── Post.php       # Класс статьи
│       └── Comment.php    # Класс комментария
├── composer.json          # Конфигурация Composer с PSR-4 автозагрузкой
├── index.php             # Веб-интерфейс
├── styles.css            # Стили интерфейса
└── start-server.bat      # Скрипт для запуска сервера
```

## Особенности

- **PSR-4 автозагрузка**: Классы автоматически загружаются через Composer
- **Faker**: Используется для генерации тестовых данных
- **Веб-интерфейс**: Современный и адаптивный интерфейс для просмотра данных
- **Вкладки**: Удобная навигация между пользователями, статьями и комментариями

## Классы

### User
- `id` - идентификатор пользователя
- `firstName` - имя
- `lastName` - фамилия

### Post
- `id` - идентификатор статьи
- `authorId` - идентификатор автора
- `title` - заголовок
- `text` - текст статьи

### Comment
- `id` - идентификатор комментария
- `authorId` - идентификатор автора
- `postId` - идентификатор статьи
- `text` - текст комментария
