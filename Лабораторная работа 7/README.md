# Лабораторная работа 7 - Система лайков

## Описание

Лабораторная работа 7 реализует систему лайков для статей и комментариев.

## Функциональность

1. **Класс Like** - описывает лайк статьи со свойствами:
   - UUID лайка
   - UUID статьи
   - UUID пользователя

2. **Класс CommentLike** - описывает лайк комментария со свойствами:
   - UUID лайка
   - UUID комментария
   - UUID пользователя

3. **Таблицы в БД**:
   - `post_likes` - для хранения лайков статей
   - `comment_likes` - для хранения лайков комментариев

4. **Репозитории**:
   - `LikesRepository` - для работы с лайками статей
   - `CommentLikesRepository` - для работы с лайками комментариев

5. **HTTP Endpoints**:
   - `POST /api/posts/like.php` - добавление лайка к статье
   - `POST /api/comments/like.php` - добавление лайка к комментарию
